# YouTube Clone - Frontend UI 📺

A pixel-perfect, responsive frontend replica of the YouTube homepage. This project was built to master complex web layouts using modern CSS techniques without the use of frameworks.

## 🚀 Live Demo
**Check it out here:** [Insert your Netlify or GitHub Pages Link Here]

## 🛠️ Built With
- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom styling with a focus on:
  - **Flexbox & CSS Grid:** For the navigation bar and responsive video thumbnail grid.
  - **Positioning:** Fixed sidebars and headers.
  - **Hover Effects:** Interactive states for buttons and thumbnails.

## 🧠 What I Learned
While building this project, I tackled several challenges:
- **Responsive Design:** Using media queries to ensure the grid shifts from 4 columns to 1 column based on screen size.
- **Icon Integration:** Using external font/icon libraries to match the real YouTube UI.
- **Layout Logic:** Managing nested divs to keep the code organized and scalable.


## 🏁 How to Run
1. Clone the repository: 
   `git clone https://github.com`
2. Open `youtube_clone.html` in your favorite browser.
